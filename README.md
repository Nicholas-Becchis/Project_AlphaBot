# Project_AlphaBot
Client and server(TCP)  which allows you to control the alphabot robot, thanks also to the use of an sqlite database


## API Reference

### Simple Movements

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `api_key` | `string` | **Required**. Your API key |
